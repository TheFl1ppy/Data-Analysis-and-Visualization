# Анализ и визуализация данных (EDA)

Анализ данных о заболеваемости с использованием Pandas, NumPy, Matplotlib и Seaborn.

---

## О проекте

Проект демонстрирует полный цикл работы с данными:
- загрузка и объединение датасетов
- очистка и обработка пропусков
- анализ распределений и поиск зависимостей
- визуализация ключевых закономерностей

---

## Стек технологий

| Инструмент | Назначение |
|------------|------------|
| **Python** | Язык программирования |
| **Pandas** | Обработка и анализ данных |
| **NumPy** | Работа с массивами и вычислениями |
| **Matplotlib** | Базовая визуализация |
| **Seaborn** | Статистическая визуализация |
| **Jupyter Notebook** | Интерактивная среда разработки |

---

## Данные

Использовались два датасета:
- `features_train.csv` — признаки (метеоданные, географические характеристики)
- `labels_train.csv` — целевая переменная (`total_cases` — количество случаев заболевания)

Данные содержат информацию по городам, неделям года, температуре и другим метеопараметрам.

---

## Этапы анализа

### 1. Загрузка и объединение данных
- Загрузка двух CSV-файлов с помощью `pd.read_csv()`
- Объединение датасетов по общему ключу через `merge()`

### 2. Первичный анализ данных
- Просмотр структуры (`info()`, `describe()`)
- Выявление пропусков и аномалий

### 3. Очистка данных
- Заполнение пропусков нулевыми значениями (`fillna(0)`)
- Удаление столбцов с большим количеством пропусков и нерелевантных признаков:
  - `ndvi_ne`, `ndvi_nw`, `ndvi_se`, `ndvi_sw` (индексы растительности)
  - температурные показатели (`reanalysis_*_temp_k`)
- Удаление строк с оставшимися пропусками (`dropna()`)

### 4. Преобразование данных
- Конвертация столбцов с датами в формат `datetime`

### 5. Визуализация
- Распределение целевой переменной (`displot`)
- Зависимость числа случаев от даты (`displot`)
- Корреляция температуры и недели года (`lmplot`)
- Динамика заболеваемости по неделям (`barplot`)
- Распределение случаев по городам (`plot`)
